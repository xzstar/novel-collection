# 第8章 泉滴古树

第3卷 第8章 泉滴古树

金光一闪，叶星身形自泉眼处掠出，穿过层层通道，重返树心大厅。

大厅内，那株枯萎的树心光球已黯淡至几近熄灭，只剩一缕微弱的绿芒在核心处跳动，如风中残烛。叶星刚踏入大厅，古树残魂的虚影便从光球中浮现，面容比之前更加枯槁，身形几近透明。

“年轻人……你回来了……”残魂的声音虚弱至极，却带着一丝急切，“泉水……可曾到手？”

叶星点头，从怀中取出玉瓶。瓶身通透，内中盛满翠绿色的泉水，散发出圣洁的生机气息。那气息一出现，大厅内的恶念雾气竟如遇克星般退散数尺。

“好！好！”古树残魂眼中闪过欣慰之色，“快……将泉水滴在根部……时间不多了……”

叶星却未立即行动，沉声问道：“前辈，灭世之魔的封印……还能支撑多久？”

古树残魂沉默片刻，缓缓道：“封印已出现裂缝……血色光柱冲天而起，便是封印崩溃的征兆。老夫感知……最多只剩一日。”

“一日？”叶星心头一沉。

“不错。”古树残魂叹息，“千年封印，本已摇摇欲坠。你破解黑色晶壁，释放净化之泉，虽是大功，却也加速了封印的崩溃——那晶壁本就是封印的一部分，用以压制灭世之魔的恶念外泄。如今晶壁破碎，恶念再无束缚，封印崩溃的速度……比老夫预想的快了数倍。”

叶星握紧玉瓶，眼中闪过决然之色：“那便不能再耽搁了。”

他转身，朝树心大厅深处走去。

古树根部位于大厅正中央，那里有一块巨大的青石台，石台上布满密密麻麻的树根，根须虬结如龙蛇，却已枯黄干裂，毫无生机。更令人心悸的是，那些树根上缠绕着黑色的恶念雾气，正不断侵蚀着古树的根基。

叶星走到石台前，单膝跪地，双手捧起玉瓶。

“前辈，晚辈这便滴下泉水。”

古树残魂的虚影悬浮在他身侧，声音庄重而肃穆：“年轻人，你要记住——净化之泉乃是生命古树的本源灵液，一滴便可催生一片森林。你将泉水滴在根部，古树便会复苏，净化恶念，修复根基。但复苏之时，必会引来灭世之魔的注意……他绝不会坐视古树恢复。”

“晚辈明白。”叶星深吸一口气，拔开瓶塞。

翠绿色的泉水在瓶中荡漾，散发出柔和的金色光芒。那光芒温暖而圣洁，仿佛能洗涤世间一切污秽。叶星小心翼翼地将玉瓶倾斜，一滴泉水从瓶口滑落——

“滴答。”

泉水落在枯黄的树根上，发出一声清脆的声响。

那一瞬间，时间仿佛凝固。

泉水渗入树根，金色的光芒从落点处蔓延开来，如涟漪般向四周扩散。枯黄的树根在金光的浸润下，竟以肉眼可见的速度恢复生机——干裂的树皮重新变得光滑，枯死的根须重新长出嫩芽，黑色的恶念雾气在金光的照耀下如冰雪般消融。

“嗡——”

整棵生命古树微微震颤，一股浩瀚的生机从根部涌出，沿着树干向上蔓延。叶星抬头望去，只见大厅顶部的枝叶开始焕发出翠绿色的光芒，那些枯萎了千年的树枝，竟重新长出了嫩叶。

“成功了……”古树残魂的声音带着哽咽，“千年了……生命古树……终于复苏了……”

叶星站起身，看着眼前的变化，心中涌起一股难以言喻的欣慰。然而就在这时——

“轰隆！”

一声巨响从古树外传来。

叶星脸色一变，身形一闪，掠出树心大厅，来到古树外部。

眼前的景象，让他瞳孔骤缩。

天空之中，那道冲天而起的血色光柱正在剧烈颤动，光柱表面浮现出一道道裂纹，仿佛有什么东西正在从内部冲击封印。更令人心悸的是，光柱的裂缝中，竟传出一阵低沉的笑声——

“呵呵呵……蝼蚁……你以为净化一棵树，就能阻止我？”

那声音沙哑而低沉，仿佛从九幽深处传来，带着令人窒息的威压。叶星只觉心脏猛地一跳，怀中的黑色晶石碎片竟开始发热，与那笑声产生共鸣。

“这是……灭世之魔的意志！”叶星咬牙，强行压下心中的悸动。

血色光柱的裂缝越来越大，一道模糊的黑影从裂缝中浮现，俯瞰着下方的生命古树。那黑影没有具体的形态，只是一团扭曲的黑暗，却散发出令人绝望的气息。

“虚空印记的继承者……”黑影的声音在叶星识海中响起，“你可知，你做的这一切，不过是徒劳？”

“封印已破，本座即将苏醒。届时，这片星域将化为血海，所有生灵都将成为本座的养料。”

“而你——将成为本座复苏后的第一个祭品！”

话音未落，黑影抬手，一道血色光柱从天而降，直轰向生命古树！

叶星脸色大变，正要出手抵挡——

“嗡！”

生命古树突然爆发出耀眼的金色光芒，一道光罩从树冠升起，将血色光柱挡在外面。两股力量碰撞，爆发出震耳欲聋的轰鸣，空间都在剧烈震颤。

“哼……垂死挣扎。”黑影冷哼一声，身形缓缓消散，“本座倒要看看，这棵破树能撑多久……”

话音未落，血色光柱恢复平静，裂缝也渐渐合拢。但叶星知道，这只是暂时的平静——灭世之魔的意志已经苏醒，真正的危机，才刚刚开始。

他深吸一口气，转身返回树心大厅。

大厅内，古树残魂的虚影比之前凝实了许多，显然古树的复苏让他恢复了一些力量。但此刻，他的脸上却满是凝重之色。

“年轻人，你看到了。”古树残魂沉声道，“灭世之魔的意志已经苏醒，他随时可能冲破封印。”

“前辈，可有办法彻底封印他？”叶星问道。

古树残魂沉默片刻，缓缓道：“有。但需要你去做一件事。”

“什么事？”

“找到钥匙碎片。”古树残魂抬手，指向叶星怀中的黑色晶石碎片，“你身上的这块碎片，便是封印钥匙的一部分。当年星宫之主将钥匙一分为三，分别藏在古战场的三个地方——你手中的这块，来自净化之泉的封印。另外两块，则藏在‘葬魂渊’和‘星陨谷’。”

“只有集齐三块碎片，才能重新激活封印，将灭世之魔彻底镇压。”

叶星握紧怀中的黑色晶石碎片，眼中闪过决然之色：“葬魂渊……星陨谷……晚辈这便前往。”

“且慢。”古树残魂叫住他，“葬魂渊和星陨谷皆是古战场的险地，恶念生物横行，更有灭世之魔的爪牙镇守。你一人前往，凶多吉少。”

“晚辈别无选择。”叶星沉声道，“若让灭世之魔苏醒，整个星域都将化为血海。晚辈虽修为低微，却也不愿坐以待毙。”

古树残魂盯着他，沉默良久，缓缓点头：“好。既然你心意已决，老夫便送你一程。”

他抬手，掌心浮现出一道金色的符文，朝叶星飞去。

“这是生命古树的印记，可助你感应钥匙碎片的位置。另外……”古树残魂顿了顿，眼中闪过复杂之色，“你要小心……葬魂渊和星陨谷中，有灭世之魔的两位护法镇守。他们的实力，远非暗河巨兽可比。”

叶星接过金色符文，郑重抱拳：“多谢前辈提醒。”

他转身，正要离开——

“小心！他来了……”

古树残魂的声音突然变得急促而惊恐。

叶星心头一凛，正要询问——

“轰！”

一道黑影从血色光柱中降临，直扑叶星！

那黑影速度极快，眨眼间便到了叶星面前，一只漆黑的手掌朝他胸口抓来。手掌上缠绕着浓郁的血色符文，散发出令人作呕的恶念气息。

叶星脸色大变，身形急退，同时抬手——

“虚空——斩！”

一道金色刀芒从掌心斩出，与那黑影的手掌碰撞在一起。

“轰隆！”

两股力量碰撞，爆发出剧烈的冲击波。叶星只觉胸口一闷，整个人被震得倒飞出去，重重撞在树心大厅的墙壁上。

“噗——”一口鲜血喷出。

那黑影却纹丝不动，悬浮在半空中，俯瞰着叶星。

“虚空斩……倒是有点意思。”黑影缓缓开口，声音沙哑而低沉，“不过，以你现在的修为，连本座的一根手指都伤不了。”

叶星挣扎着站起身，擦去嘴角的血迹，眼中闪过凝重之色。

这黑影的实力……远超他的想象。

“你是……灭世之魔的护法？”叶星沉声道。

黑影没有回答，只是缓缓抬起手，掌心浮现出一道血色符文。

“小子，本座奉主人之命，前来取你性命。”

“受死吧！”
