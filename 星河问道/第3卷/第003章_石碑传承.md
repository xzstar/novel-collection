# 第3章 石碑传承

石碑传承

金光如瀑，自石碑中倾泻而下，化作一道屏障，将恶念分身震退数丈。

“什么？！”

恶念分身落地，脸色阴沉如水。他盯着那道金色屏障，眼中闪过忌惮之色。

叶星也是一惊，抬头望向石碑。

石碑上的符文缓缓流转，一道苍老的声音自碑中传出：

“虚空印记的传承者……你终于来了。”

话音未落，一道虚影自石碑中浮现。

那是一位老者，须发皆白，身着青色道袍，周身缭绕着淡淡的金色光晕。他的面容模糊不清，但那双眼睛却格外明亮——仿佛能洞穿虚空，看透一切虚妄。

“你是……”叶星警惕地问道。

老者虚影微微一笑：“老夫乃生命古树守护者，奉命在此等候虚空印记的传承者。”

“生命古树守护者？”叶星心头一震，“这石碑……与生命古树有关？”

“不错。”老者虚影点头，“此碑乃生命古树以自身枝干所铸，封印着虚空印记的完整传承。唯有虚空印记的持有者，方能开启。”

恶念分身闻言，眼中闪过狰狞之色：“区区传承，也敢阻我？等我吞噬了这心脏，你们都得死！”

他再次化作一道血光，朝石碑扑来。

老者虚影抬手一挥，金色屏障骤然亮起，化作一道光幕，将恶念分身挡在外面。

“轰——”

恶念分身撞在光幕上，发出一声闷响。金光震荡，一股强大的反噬之力涌出，将他震飞出去。

“噗——”

恶念分身喷出一口鲜血，眼中闪过惊怒之色：“这屏障……与心脏相连？”

“不错。”老者虚影淡淡道，“此屏障以心脏之力为基，你若强行破解，只会引爆心脏空间。届时，你我皆会葬身于此。”

恶念分身脸色一变，咬牙道：“算你狠！”

他退后几步，死死盯着叶星，眼中闪过不甘之色。

老者虚影看向叶星，眼中闪过一丝赞许：“小辈，你可愿接受虚空传承的考验？”

叶星深吸一口气，拱手道：“晚辈愿意。”

“好。”老者虚影点头，“传承共分三关。第一关，心魔幻境——考验你的意志。若通不过，轻则修为尽废，重则神魂俱灭。你可想清楚了？”

叶星握紧灵剑，眼中闪过决然之色：“晚辈心意已决。”

“很好。”老者虚影抬手一挥，一道金光将叶星笼罩，“那就开始吧。”

话音未落，叶星眼前一黑，仿佛坠入无尽深渊。

等他再睁开眼时，发现自己置身于一片熟悉的场景——

那是他曾经的家园。

青山绿水，鸟语花香。父母在田间劳作，弟弟在溪边嬉戏。一切都那么美好，仿佛从未发生过变故。

“星儿，回来吃饭了！”母亲的声音从远处传来。

叶星心头一颤，眼中闪过痛苦之色。

他知道，这是幻境。

但他还是忍不住朝那个方向走去。

然而就在这时——

天空骤然暗了下来。

一道血色裂缝出现在天际，无数黑影从裂缝中涌出，扑向村庄。

“啊——”

惨叫声此起彼伏。

叶星看到父母倒在血泊中，弟弟被黑影吞噬，宗门覆灭，师兄弟惨死……

“不——”

他怒吼一声，想要冲过去，却发现身体被无形的力量束缚，动弹不得。

“看到了吗？”一道阴冷的声音在耳边响起，“这就是你的软弱，你的无能。你救不了他们，你什么都做不了。”

一道黑影从虚空中走出——正是恶念分身。

他嘴角挂着讥讽的笑容：“放弃吧，叶星。你太弱了，根本不配拥有虚空印记。把力量交给我，让我来替你复仇。”

叶星咬紧牙关，眼中闪过愤怒之色。

但他没有出手。

因为他知道，这是心魔。

一旦被愤怒冲昏头脑，就会陷入幻境，永世不得超脱。

“我不管你是谁，既然选择了我，我就不会辜负这份力量！”

叶星闭上双眼，催动识海中的虚空印记。

淡金色的光芒自他体内涌出，化作一道光柱，冲天而起。

“破！”

他猛然睁开双眼，眼中闪过一道金光。

眼前的幻境瞬间破碎，化作无数碎片，消散在虚空中。

叶星发现自己回到了石碑前。

老者虚影眼中闪过赞许之色：“不错，能在如此短的时间内识破心魔，你的意志远超老夫预料。”

叶星深吸一口气，拱手道：“前辈过奖了。”

“第一关，你通过了。”老者虚影抬手一挥，一道金光没入叶星体内，“接下来，是第二关——虚空法则领悟。”

话音未落，叶星感到一股强大的吸力，将他拉入一片奇异的空间。

这里没有天地，没有方向，只有无数符文在虚空中流转。

那些符文闪烁着淡金色的光芒，仿佛在演绎某种至高法则。

叶星心头一震，立刻盘膝坐下，闭目感悟。

符文在他周围旋转，化作一道道流光，没入他的识海。

虚空印记微微颤动，与那些符文产生共鸣。

叶星感到自己对虚空法则的理解，正在飞速提升。

空间切割、虚空穿梭、时空扭曲……无数法则奥义在他脑海中浮现。

他仿佛看到了宇宙的起源，看到了虚空的本质。

“原来如此……”

叶星喃喃自语，眼中闪过明悟之色。

就在这时，石碑之灵的声音在他耳边响起：

“小辈，你的天赋远超老夫预料。但时间不多了，那恶念分身正在试图破解屏障。”

叶星睁开眼，眼中闪过一道金光：“前辈，我准备好了。”

“好。”石碑之灵的声音带着一丝欣慰，“那就开始第三关——传承融合。”

话音未落，一股浩瀚的力量自石碑中涌出，涌入叶星体内。

叶星感到自己的身体仿佛要被撑爆，无数法则奥义在他体内碰撞、融合。

虚空印记剧烈震颤，散发出刺目的金光。

“啊——”

叶星发出一声痛呼，额头青筋暴起。

就在这时——

“轰！”

一声巨响传来。

恶念分身终于按捺不住，开始疯狂攻击屏障。

“该死！给我破！”

他化作一道血光，不断撞击金色屏障。

屏障剧烈震颤，表面浮现出一道道裂纹。

“不好！”石碑之灵脸色一变，“这恶念分身疯了，他想要同归于尽！”

叶星咬牙，强忍着体内的剧痛：“前辈，还能撑多久？”

“最多半柱香。”石碑之灵沉声道，“若无法完成传承，我们都会葬身于此。”

叶星深吸一口气，闭上双眼，全力催动虚空印记。

金色的光芒在他体内流转，与石碑中的传承之力融合。

时间一分一秒过去。

恶念分身的攻击越来越猛烈，屏障上的裂纹越来越多。

“给我破！”

恶念分身怒吼一声，化作一道血光，狠狠撞在屏障上。

“轰——”

屏障终于破碎，化作无数碎片，消散在虚空中。

恶念分身落地，眼中闪过狰狞之色：“哈哈哈哈哈！传承是我的了！”

然而就在这时——

石碑骤然爆发出刺目的金光，将整个空间照亮。

一股浩瀚的力量自石碑中涌出，涌入叶星体内。

“什么？！”恶念分身脸色大变，“你……你完成了传承？！”

叶星睁开双眼，眼中闪过一道金光。

他感到体内充满了力量，虚空印记已经彻底与他的神魂融合。

“不错。”叶星淡淡道，“多谢你帮我拖延时间。”

恶念分身脸色阴沉如水：“该死！我要杀了你！”

他化作一道血光，朝叶星扑来。

叶星抬手一挥，一道金色剑光斩出。

“虚空斩！”

剑光所过之处，空间被撕裂，露出漆黑的虚空裂缝。

恶念分身脸色大变，想要躲避，却已经来不及。

“噗——”

剑光斩在他身上，留下一道深可见骨的伤口。

“啊——”恶念分身发出一声惨叫，倒飞出去。

叶星落地，握紧灵剑，冷冷道：“现在，该结束了。”

然而就在这时——

石碑之灵的声音在他耳边响起：

“小辈，记住……心脏是钥匙，也是牢笼……摧毁它，才能终结一切。”

话音未落，石碑之灵的身影渐渐消散，化作点点金光，融入虚空。

叶星心头一震：“前辈！”

但石碑之灵已经彻底消散，只留下一句话：

“去吧……完成你的使命……”

叶星握紧拳头，眼中闪过坚定之色。

他看向恶念分身，冷冷道：“今日，便是你的死期。”

恶念分身挣扎着站起身，眼中闪过疯狂之色：“想杀我？那就一起死！”

他伸手按在心脏上，催动体内的力量。

心脏剧烈震颤，表面浮现出一道道裂纹。

“不好！”叶星脸色一变，“他要引爆心脏！”

恶念分身狂笑：“哈哈哈哈哈！既然我得不到，那就谁也别想得到！”

心脏开始剧烈震动，裂缝蔓延，整个空间都在崩塌。

叶星咬牙，握紧灵剑，眼中闪过决然之色。

摧毁心脏，与恶念分身同归于尽？

还是另寻他法？

时间不多了。
