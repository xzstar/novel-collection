# 第5章 古战之秘

第3卷 第5章 古战之秘

时空乱流如刀，割裂着古战场深处的虚空。

叶星手握黑色晶石，沿着那缕金色光芒的指引，在乱流中艰难前行。晶石表面流转的符文微微震颤，发出低沉的共鸣声，仿佛在回应某种召唤。

四周的时空碎片不断变幻，时而浮现出远古战场的惨烈画面——无数修士在虚空中厮杀，灵光如雨，血染星河；时而又化作一片死寂的虚空，只有残破的星骸漂浮其中。

叶星屏息凝神，避开一道从侧面袭来的时空裂缝。那裂缝掠过之处，空间被撕开一道漆黑的裂口，久久不能愈合。

“此地法则之混乱，远超外界……”他低声自语，眼中闪过凝重之色。

黑色晶石突然发出一阵急促的震颤。

叶星脚步一顿，抬头望去。

前方，时空乱流之中，一座残破的宫殿遗迹静静矗立。

那宫殿通体由暗金色的星石砌成，表面刻满密密麻麻的符文。大部分符文已被岁月磨蚀，但仍有少数几道纹路闪烁着微弱的灵光。宫殿的穹顶已经坍塌，只剩下四壁和一座残破的祭坛，在时空乱流中摇摇欲坠。

“这里……就是牢笼的入口？”

叶星握紧晶石，缓步朝宫殿走去。

越靠近宫殿，时空乱流就越发狂暴。无数残魂从碎片中冲出，发出无声的咆哮，试图阻拦他的去路。叶星催动虚空印记，周身泛起淡金色的光芒，将残魂逼退。

终于，他踏上了宫殿的台阶。

脚下的星石传来一阵冰凉，仿佛踩在万年寒冰之上。叶星抬头，目光扫过宫殿四壁——那些壁画虽然残破，但仍能看出大致内容。

第一幅壁画上，一位身披星袍的修士站在虚空之中，双手结印，将一团漆黑的魔影封印在一颗心脏中。

第二幅壁画上，那心脏被放置在祭坛上，周围站着数十位修士，同时施展封印术，将心脏镇压在古战场深处。

第三幅壁画上，一颗巨大的古树虚影笼罩着整个封印之地，无数根须深入虚空，将心脏牢牢锁住。

“这是……星宫之主封印灭世之魔的过程？”叶星喃喃道，眼中闪过震惊之色。

他继续往前走，在祭坛边缘发现了一段文字。

那文字以古篆书写，笔画苍劲有力，带着岁月沉淀的厚重感：

“吾以毕生修为，封印灭世之魔于心脏之中。此心为牢，亦为钥。唯有虚空印记之继承者，方能开启封印，彻底灭杀此魔。”

“然，若继承者心志不坚，被恶念侵蚀，则封印反噬，魔将破牢而出，吞噬诸天。”

“故，吾留此碑于心脏之内，以生命古树之力镇守，唯有通过三关考验者，方可获得传承。”

叶星读完，心中掀起惊涛骇浪。

“原来……那颗心脏既是牢笼，也是钥匙。”他低头看向手中的黑色晶石，眼中闪过复杂之色，“星宫之主留下传承，是为了让继承者彻底灭杀灭世之魔。但若继承者被恶念侵蚀，反而会助魔脱困……”

他正想着，突然——

一股恐怖的威压从祭坛中央爆发！

叶星脸色大变，身形暴退。

一道黑影从祭坛上缓缓浮现，逐渐凝聚成一道人形虚影。那虚影身披星袍，面容模糊，只有一双眼睛闪烁着幽冷的光芒，死死盯着叶星。

“虚空印记的继承者……你身上为何有恶念的臭味？”

那声音苍老而威严，带着难以言喻的压迫感，仿佛来自远古时代。

叶星心头一凛，握紧黑色晶石，沉声道：“晚辈叶星，奉石碑之灵指引，前来探寻牢笼之秘。恶念气息乃封印恶念分身时所留，并非晚辈自身所染。”

“封印恶念分身？”黑影冷笑一声，“你可知那恶念分身，乃是星宫之主斩下的恶念所化？你以寿元封印它，虽暂时压制，但恶念气息已沾染你的神魂。若不及早净化，终有一日，你会被恶念侵蚀，沦为灭世之魔的傀儡！”

叶星脸色微变，但很快恢复平静。

他深吸一口气，抬起右手，掌心浮现出一道金色的虚空印记。

“晚辈以虚空印记自证清白。”他沉声道，“若晚辈已被恶念侵蚀，虚空印记必然黯淡无光。但此刻，印记依旧纯净如初，足以证明晚辈心志坚定，未被污染。”

黑影盯着那道虚空印记，沉默片刻。

突然，他身形一闪，化作一道流光，朝叶星扑来！

“既然你自称清白，那就让本座亲自验证！”

黑影一掌拍出，掌风如刀，撕裂虚空。

叶星瞳孔骤缩，想也不想，催动虚空印记，身形一闪，消失在原地。

下一瞬，他出现在三丈之外，但黑影如影随形，又是一掌拍来。

“躲得倒快。”黑影冷笑，“但你以为，仅凭虚空印记就能逃过本座的感知？”

叶星咬牙，知道躲不过去，当即催动星灵气，右手结印，施展虚空斩！

“虚空——斩！”

一道金色刀芒从掌心斩出，撕裂虚空，朝黑影劈去。

黑影眼中闪过一丝惊讶，身形微侧，避开刀芒。但那刀芒掠过之处，空间竟被撕开一道细小的裂痕，久久不能愈合。

“虚空斩……你竟领悟了这一式。”黑影停下攻击，盯着叶星，“看来，你的确得到了虚空传承。”

叶星收招，微微喘息，面色苍白如纸。

燃烧寿元的后遗症还未完全恢复，此刻强行施展虚空斩，让他感到一阵虚弱。

“晚辈以寿元为代价，封印恶念分身，自证清白。”他沉声道，“若前辈不信，尽管一试。”

黑影沉默片刻，缓缓开口：“你的确未被彻底污染。但你的根基已损，若不修复，连承受封印之力的资格都没有。”

叶星一怔：“前辈的意思是……”

“生命古树。”黑影淡淡道，“你可知那石碑之灵，便是生命古树的守护者？”

叶星心头一震，想起石碑之灵临死前的话：“老朽乃生命古树守护者，在此等候虚空印记的继承者……”

“生命古树……到底是什么？”他忍不住问道。

黑影沉默片刻，缓缓道：“生命古树，乃是宇宙初开时诞生的第一棵灵树。它扎根于虚空之中，汲取星灵气，孕育万物。星宫之主当年封印灭世之魔，便是借助生命古树的力量，将心脏镇压在古战场深处。”

“但千年前，生命古树被恶念侵蚀，逐渐枯萎。守护者不得不将虚空传承封印在石碑中，等待有缘人。”

叶星听得心惊肉跳，忍不住问道：“那生命古树现在何处？”

黑影抬手，指向宫殿深处：“穿过祭坛，便是生命古树的入口。但那里早已不是圣地——恶念侵蚀之下，古树内部已成险境。你若要去，需先找到‘净化之泉’的坐标，否则，你连古树外围都进不去。”

“净化之泉？”叶星皱眉，“那是什么？”

“生命古树的核心灵泉，可以净化恶念，修复根基。”黑影道，“但它的坐标，早已失传千年。只有古树内部的守护者，才知道它的下落。”

叶星沉默片刻，眼中闪过决然之色。

“晚辈愿意前往生命古树，修复根基，完成封印。”

黑影盯着他，沉默良久，缓缓点头：“好。既然你心意已决，本座便送你一程。”

他抬手，掌心浮现出一道金色的符文，朝叶星飞去。

叶星伸手接住，那符文化作一道流光，没入他的眉心。

“这是生命古树的坐标。”黑影道，“但本座要提醒你——古树内部，不仅有恶念侵蚀，还有无数被污染的守护者。你若进去，九死一生。”

叶星抱拳：“多谢前辈提醒。晚辈自有分寸。”

黑影微微点头，身形开始消散。

“去吧……时间不多了。”

话音未落，宫殿突然剧烈震颤！

四壁上的符文开始崩裂，无数时空裂缝从祭坛中央蔓延开来，将整个宫殿撕碎。

叶星脸色大变，催动虚空印记，想要逃离。

但就在这时——

手中的黑色晶石突然滚烫，发出一阵刺目的血光！

叶星低头，只见晶石表面浮现出一行血字：

“三天后，灭世之魔将苏醒。”

叶星瞳孔骤缩，心中掀起惊涛骇浪。

“三天……只有三天了？”

他来不及多想，身形一闪，消失在原地。

下一瞬，他被时空乱流卷出，重重摔在古战场裂缝的入口处。

“咳咳……”

叶星挣扎着爬起，低头看向手中的黑色晶石。

那行血字已经消失，但晶石依旧滚烫，仿佛在提醒他时间的紧迫。

“三天……我必须在这三天内，找到净化之泉，修复根基，完成封印。”

他深吸一口气，眼中闪过决然之色。

“否则，灭世之魔一旦苏醒，整个星域都将陷入浩劫……”

话音未落，他身形一闪，消失在原地，朝生命古树的坐标飞去。
